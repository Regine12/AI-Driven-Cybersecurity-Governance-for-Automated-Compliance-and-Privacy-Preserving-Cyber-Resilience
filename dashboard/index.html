> ABOUT THIS PROJECT

> ABOUT_

AI-DRIVEN CYBERSECURITY GOVERNANCE

This prototype is part of a Master's thesis in the CyberMACS — Applied Cybersecurity programme, demonstrating how selected requirements from multiple regulatory frameworks can be translated into structured, machine-interpretable governance logic for automated and continuous compliance monitoring.

The real contribution is a governance framework that maps controls across ISO/IEC 27001, NIST CSF 2.0, Cyber Essentials and GDPR into a unified model, enabling cross-standard compliance assessment through a lightweight rule engine.

GOVERNANCE FOCUSED

Management-level framework, not a deep technical implementation.

TRACEABILITY

End-to-end auditability from requirement source to compliance outcome.

AUTOMATED

Machine-interpretable rules for continuous, repeatable assessment.
